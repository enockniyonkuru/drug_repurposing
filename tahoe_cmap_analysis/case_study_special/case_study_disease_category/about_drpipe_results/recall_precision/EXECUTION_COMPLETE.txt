DATA NOTES
================================================================================

Data Sources:
  - CMAP recovered: 948 validated disease-drug pairs
  - TAHOE recovered: 2,198 validated disease-drug pairs
  - CMAP all discoveries: 5,241 predicted pairs
  - TAHOE all discoveries: 9,946 predicted pairs
  - Open Targets: Reference validation database

Metrics Calculated:
  - Precision = Successfully recovered / All predictions
  - Recall = Successfully recovered / Maximum possible
  - Maximum possible = Known drugs available in platform universe

Statistical Tests:
  - T-tests for platform comparison
  - Percentile analysis (P5, Q1, Median, Q3, P95)
  - Distribution shape (skewness, kurtosis)
