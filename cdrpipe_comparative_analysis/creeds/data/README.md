# creeds data guide

This folder contains the CREEDS-specific data used in the comparative-analysis workflow.

## Replication order

1. `raw_creeds_exports/`
2. `manual_signatures_extracted/`
3. `manual_signatures_standardized/`
4. `../results/manual_standardized_all_diseases_results/`
5. `../analysis/manual_standardized_all_diseases_analysis/`

## Data folders

### `raw_creeds_exports/`

The CREEDS export files downloaded from the web and placed into the repository manually.

### `manual_signatures_extracted/`

One `*_signature.csv` file per disease after extracting the CREEDS manual export.

Generated by:

- [process_creeds_signatures.py](/Users/enockniyonkuru/Desktop/drug_repurposing/cdrpipe_comparative_analysis/creeds/scripts/processing/process_creeds_signatures.py)

### `manual_signatures_standardized/`

The cleaned signatures used for the CREEDS batch drug-repurposing runs.

Generated by:

- [standardize_creeds_signatures.py](/Users/enockniyonkuru/Desktop/drug_repurposing/cdrpipe_comparative_analysis/creeds/scripts/processing/standardize_creeds_signatures.py)

### `manual_signatures_shared_genes/`

The CREEDS signatures filtered to the gene universe shared by CMAP and TAHOE.

The shared-gene reference can still be regenerated with:

- [compare_tahoe_cmap.py](/Users/enockniyonkuru/Desktop/drug_repurposing/cdrpipe_comparative_analysis/creeds/scripts/analysis/compare_tahoe_cmap.py)

The exact script that wrote this full folder is not preserved in the curated tree.

### `signature_gene_counts_across_stages.csv`
### `signature_gene_counts_across_stages.xlsx`

Summary tables used by CREEDS-specific analysis and figure scripts.

### `disease_metadata.parquet`

Preserved reference table included for traceability.

## Running the CREEDS workflow

From `cdrpipe_comparative_analysis/`:

```bash
python creeds/scripts/processing/process_creeds_signatures.py
python creeds/scripts/processing/standardize_creeds_signatures.py
Rscript scripts/execute/run_batch_from_config.R \
  --config_file creeds/scripts/execute/creeds_manual_config_all_avg.yml
python creeds/scripts/analysis/extract_pipeline_results_analysis.py \
  --input_dir creeds/results/manual_standardized_all_diseases_results \
  --output_dir creeds/analysis/manual_standardized_all_diseases_analysis
```

The canonical workbook produced by this workflow is:

- [CREEDS_Manual_All_Diseases_Analysis.xlsx](/Users/enockniyonkuru/Desktop/drug_repurposing/cdrpipe_comparative_analysis/creeds/analysis/CREEDS_Manual_All_Diseases_Analysis.xlsx)
